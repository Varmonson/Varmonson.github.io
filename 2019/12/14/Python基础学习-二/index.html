Python基础学习(二)

发表于 2019-12-14 | 分类于 Python | 阅读次数:
字数统计: 1.7k

这篇我们开始学习Python的基本数据类型：数字类型和字符串类型。

1.数字类型

Python中的数字类型有三种：整数、浮点数、复数。

1.1整数类型

Python中的整数类型没有大小限制

1.2浮点数

Python中的浮点数表示区间范围内的实数，取值范围数量级约-10^307至10^308，精度数量级10^-16，所以可以当作基本没有限制。还有一点，对于浮点数的运算，存在不确定尾数，因为计算机中运算是转化为二进制运算的，对于一些数值在转化过程中会无法整除2，所以是近似等于。

1.3复数类型

Python中还有一种复数的数字类型，这在大多数编程语言中是没有的，所以我们可以很好的利用Python进行数学建模。这里用到的pow(x,y)函数下面会提到。

1.4数值运算操作符

运算符 说明
x + y 表示x加y的值
x - y 表示x减y的值
x * y 表示x乘以y的值
x / y 表示x除以y的值，结果是一个浮点数
x // y 表示x整除y的值，结果是一个整数
x % y 表示x对y取模
x ** y 幂运算，表示x的y次方

数值类型可以进行混合运算，计算结果为”最宽”的类型：整数 > 浮点数 > 复数

1.5数值运算函数

函数 说明
abs(x) 求绝对值
divmod(x, y) 求商余，同时输出商和余数
pow(x, y[,z]) 幂余，(x**y)%z，[..]表示参数z可省略
round(x[,d]) 四舍五入，d是保留小数位数，默认值为0
max(x1, x2…,xn) 最大值，返回x1,x2…,xn中的最大值，n不限
min(x1, x2…,xn) 最小值，返回x1,x2…,xn中的最小值，n不限
int(x) 将x变成整数，舍弃小数部分
float(x) 将x变成浮点数，增加小数部分
complex(x) 将x变成复数，增加虚数部分

2.字符串类型

无论是在哪种编程语言中，对于字符串的操作无疑是最长用的。下面我们将学习一下在Python中
对字符串的操作以及字符串函数。

2.1字符串类型的表示

有两类四种表示方法：

  • 由一对单引号或者双引号表示

  • 由一对三单引号或者三双引号表示,这种表示方式，可以跨多行文本

字符串的序号：

字符串的每一个字符所在位置为其在该字符串中的索引，我们直接通过s[i]来获取该字符串某一个索引处的字符。

字符串切片：
我们可以通过[m:n:k]根据步长对字符串切片，m表示开始索引，n表示结束索引，k表示每次的步长。

2.2字符串操作符

运算符 说明
x + y 连接两个字符串x和y
n × s 或 s × n 复制n次字符串s
x in s 判断x是否是s的字串

2.3字符串处理函数

下面是一些以函数形式提供的一下字符串处理功能。

函数 说明
len(x) 获取字符串的长度
str(x) 将任意类型转化为字符串类型
hex(x)或oct(x) 整数x的16进制或者8进制的字符串形式
chr(x) 将Unicode编码转化为字符
ord(x) 将字符转化为Unicode编码

2.4字符串处理方法

下面是字符串的一些处理方法，方法本身也是函数，但是方法是以’对象.函数’的方式使用的。

方法 说明
str.lower() 将字符串全部转化为小写
str.upper() 将字符串全部转化为大写
str.split(sep=None) 将字符串以sep进行分割(默认以’ ‘分割)，返回一个列表
str.count(sub) 统计字串sub在字符串中出现的次数
str.replace(old, new) 将字符串中的old字符替换为new字符
str.center(width[,fillchar]) 字符串str根据宽度width居中，fillchar为占位符，可选
str.strip(chars) 从字符串str中去除左右两侧chars中的字符
str.join(iter) 在iter变量中除最后一个元素，每个元素后面增加一个iter
